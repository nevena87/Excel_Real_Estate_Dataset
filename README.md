## 🏠 Real Estate Sales Dashboard (Excel Project)
### 📌 Project Overview

This project is an interactive Real Estate Sales Dashboard created in Microsoft Excel.
The dashboard analyzes property sales performance using KPIs, charts, pivot tables, slicers, and dynamic formulas.

The goal of this project is to demonstrate practical Excel skills used in data analysis and business reporting.

## 📊 Dashboard Features
- Total Revenue analysis
- Total Sold Deals tracking
- Average Sale Price calculation
- Highest Sale overview
- Monthly sales trends
- Revenue by property type
- Interactive slicers and filters
- Dynamic KPI calculations
- Pivot tables and pivot charts
  
### 🛠️ Tools & Skills Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning
- Excel Formulas
- Dashboard Design
- KPI Analysis
  
### 📈 Key Metrics
Metric	Description
- Total Revenue	Overall generated revenue
- Total Sold Deals	Number of completed sales
- Average Sale Price	Average property value
- Highest Sale	Most expensive sold property
  
### 📂 Dataset
The dataset contains real estate transaction data including:
- Properties
